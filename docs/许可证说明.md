# 许可证说明

本项目采用 `MIT（宽松开源许可）` 许可证。

之所以选这个许可证，是因为它最符合当前仓库“公开给别人参考和继续修改”的目标：

- 别人可以直接阅读、学习和复用代码
- 别人可以在自己的机器和项目里修改使用
- 别人可以继续分发修改后的版本
- 对使用方式限制较少，更容易降低参考门槛

## 你需要知道的重点

如果别人使用这个项目代码，需要遵守的核心要求其实很简单：

1. 保留原始版权声明
2. 保留 `MIT（宽松开源许可）` 许可证文本

## 这个许可证不提供什么

`MIT（宽松开源许可）` 很宽松，但它也明确说明：

- 作者不提供任何担保
- 如果使用后出现问题，责任不由作者承担

## 如果以后想换许可证

如果你未来希望“别人修改后也必须继续开源”，那时可以再评估是否改成更强约束的许可证。

但对当前这个“经验分享型”的仓库来说，`MIT（宽松开源许可）` 是一个足够合适、也最容易被理解和接受的选择。
