# P0 测试清单

- 程序可以启动
- 悬浮窗显示正常
- 悬浮窗保持置顶
- 点击按钮不抢输入焦点
- 点击 `F2` 按钮时当前焦点窗口收到 `F2`
- 点击 `F7` 按钮时当前焦点窗口收到 `F7`
- 点击 `-` 按钮时当前焦点输入框收到 `-`
- 点击 `_` 按钮时当前焦点输入框收到 `_`
- `Ctrl + 2` 触发 `F2`
- `Alt + 7` 触发 `F7`
- `Alt + 9` 输入 `-`
- `Alt + 0` 输入 `_`
- `Alt + [` 触发键盘背光降低
- `Alt + ]` 触发屏幕亮度降低
- `Ctrl + Shift + 1` 显示或隐藏悬浮窗
- 可以最小化到托盘
- 退出时不报错
- 日志文件正常生成
- 硬件控制失败时程序不崩溃
