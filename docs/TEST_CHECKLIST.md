# P0 测试清单

- 程序可以启动
- 悬浮窗显示正常
- 悬浮窗保持置顶
- 点击按钮不抢输入焦点
- 点击 `F2` 按钮时当前焦点窗口收到 `F2`
- 点击 `F7` 按钮时当前焦点窗口收到 `F7`
- 点击 `-` 按钮时当前焦点输入框收到 `-`
- 点击 `_` 按钮时当前焦点输入框收到 `_`
- 修改 `config.json` 中的 `Panel.Buttons` 后，新增按钮能在界面中显示
- 新增的 `SendKey` 自定义按钮能正确发送配置的键位或组合键
- 托盘菜单里的 `设置面板` 能正常打开
- 托盘菜单里的 `重置窗口位置到屏幕中央` 能把窗口直接拉回主屏幕可见区域
- `设置面板` 打开后不会强制长期压在其他普通窗口上方
- 在 `设置面板` 中新增、删除、调整按钮顺序后，保存可立即生效
- 在 `设置面板` 中修改按钮的 `Label`、`Action`、`Gesture`、`TriggerHotkey` 后，主界面和按钮触发行为都能立即刷新
- 在 `设置面板` 中将某个按钮的 `TriggerHotkey` 留空保存后，该按钮对应的全局触发组合键会被禁用
- 在 `设置面板` 中修改“显示/隐藏悬浮窗”快捷键后，不重启程序也能立即生效
- 当按钮数量超过 `5` 个时，界面自动换成第二排
- 当按钮数量继续增加时，界面能继续自动扩展到第三排及更多行
- `Ctrl + 2` 触发 `F2`
- `Alt + 7` 触发 `F7`
- `Alt + 9` 输入 `-`
- `Alt + 0` 输入 `_`
- `Alt + [` 触发键盘背光降低
- `Alt + ]` 触发屏幕亮度降低
- `Ctrl + Shift + 1` 显示或隐藏悬浮窗
- 可以最小化到托盘
- 退出时不报错
- 日志文件正常生成
- 硬件控制失败时程序不崩溃
